# Paradise Nursery

Paradise Nursery is a simple e-plant shopping web application built using React and Redux.

## Features
- Browse plants by category
- Add plants to shopping cart
- Update plant quantity in cart
- Remove plants from cart
- View total price of items in cart

## Technologies Used
- React
- Redux Toolkit
- JavaScript
- CSS

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
